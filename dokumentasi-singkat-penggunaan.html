CMS Sederhana

Dokumentasi CMS simple menggunakan PHP.

Pendahuluan

CMS ini terdiri dari dua bagian utama:

  1. Public Area: Tampilan depan untuk publik.
  2. Admin Area: Area admin untuk mengelola konten (tambah, edit atau hapus postingan), hanya dapat diakses dengan login.

Penjelasan struktur untuk beberapa poin penting:

  • includes/: Menyimpan file konfigurasi dan fungsi global:

    • config.php: Konfigurasi database.
    • functions.php: Fungsi-fungsi yang sering digunakan.

  • admin/: Area backend untuk mengelola konten:

    • add-content.php: Menambahkan konten.
    • edit-content.php: Mengedit konten.
    • delete-content.php: Menghapus konten.
    • index.php: Melihat semua konten.

  • public/: Area frontend yang diakses oleh publik:

    • index.php: Menampilkan daftar konten.
    • post.php: Menampilkan detail sebuah konten.

  • uploads/: Direktori untuk penyimpanan gambar yang di upload.

Instalasi

Catatan

Contoh instalasi di lokal server menggunakan Xampp.

Tempatkan source code di folder htdocs xampp dan jalankan apache server serta mysql phpmyadmin

Pembuatan Database

Membuat database dan tabel dengan kebutuhan user untuk akses dashboard admin, dan post untuk menyimpan data konten seperti judul, gambar, dsb.

pergi ke http://localhost/phpmyadmin/ > klik new > masuk ke menu SQL

  1. Buat database serta tabel user dan post:

    CREATE DATABASE my_database;
USE my_database;

CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE posts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    content TEXT NOT NULL,
    featured_image VARCHAR(255) DEFAULT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

    Ubah my_database dengan nama database yang di inginkan

  2. Buat user admin dan password pada database yang baru dibuat tadi:

    INSERT INTO users (username, password) VALUES
('admin', 'hash_password');

    NOTE: Wajib mengubah hash_password dengan hasil hash password php, bisa generate online disini

Pengaturan Awal

  1. Konfigurasi Database

    Buka file includes/config.php. Atur dan sesuaikan bagian ini dengan database yang ada:

    config.php 
    // pengaturan Database
define('DB_HOST', 'localhost');
define('DB_NAME', 'my_database');
define('DB_USER', 'root');
define('DB_PASS', '');  // sesuaikan dengan password database

  2. Pengaturan Umum

    Masih di file includes/config.php. Untuk melakukan pengaturan umum seperti merubah nama website dan lain sebagainya

    config.php 
    // pengaturan umum
    $base_url = 'http://localhost/cmsBuatan/simpleCMS/'; // wajib memberikan akhiran slash "/" diakhir url
    define('BASE_URL_SITE', $base_url . 'public/');  // URL situs
    define('BASE_URL_ADMIN', $base_url . 'admin/');  // URL admin
    define('BASE_URL_UPLOADS', $base_url . 'uploads/');  // URL direktori uploads
    define('APP_NAME', 'NamaSitus');
    define('APP_TAGLINE', 'Solusi untuk segala kehidupan Anda');

// pengaturan homepage
define('CTA_NAME', 'Info Lengkap');
define('CTA_LINK', '#about');
define('ABOUT_TEXT', '
    Text about dsb
');
define('ALAMAT_NAME', 'Nama Tempat');
define('ALAMAT_TEXT', 'alamat xxx');


// pengaturan post
define('WA_NUMBER', '628xxxx');
define('WA_TEXT', 'halo saya ingin xxx');
define('WA_LINK', 
    'https://web.whatsapp.com/send?phone=' 
    . WA_NUMBER 
    . '&text=' 
    . WA_TEXT 
);
define('WA_BTN_NAME', 'WhatsApp');

    Hal yang perlu diperhatikan, pada variable $base_url pastikan di isi dengan url tempat menjalankan code, jika di hosting maka ubah sesuai url domain misal https://yourdomain.com .

    Sisanya tinggal ubah dan sesuaikan keinginan.

    PENTING

    Ada beberapa hal yang tidak boleh di ubah untuk menghindari error, diantaranya :

    • BASE_URL_SITE
    • BASE_URL_ADMIN
    • BASE_URL_UPLOADS
    • WA_LINK

Penggunaan Dasar

Admin area

  • URL Akses: http://yourdomain.com/admin/
  • Login:
    • Masukkan username dan password untuk mengakses.
  • Menu Admin:
    • Tambah Konten:
      • Akses add-content.php.
      • Isi judul, konten, unggah gambar, dan simpan.
      • Setelah berhasil, akan diarahkan ke halaman edit konten.
    • Edit Konten:
      • Akses edit-content.php?id=<ID>.
      • Perbarui informasi konten dan simpan.
    • Hapus Konten:
      • Akses delete-content.php?id=<ID>.

  • URL Akses: http://yourdomain.com/public/
  • Menu Public:
    • Lihat Daftar Postingan:
      • Akses index.php.
    • Baca Postingan:
      • Klik judul postingan untuk membuka halaman detail (single.php?id=<ID>).
